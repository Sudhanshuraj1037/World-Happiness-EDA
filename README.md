# 📖 Project Overview

The **World Happiness Report** evaluates the happiness levels of countries using several socio-economic indicators such as:

- 💰 GDP per Capita
- ❤️ Healthy Life Expectancy
- 🤝 Social Support
- 🕊 Freedom to Make Life Choices
- 🎁 Generosity
- 🏛 Perceptions of Corruption

This project combines datasets from **2015–2019**, performs extensive data preprocessing, feature engineering, visualization, and statistical analysis to uncover trends affecting global happiness.

---

# 📂 Dataset

The project uses the official World Happiness datasets for:

- 2015
- 2016
- 2017
- 2018
- 2019

Each dataset was cleaned, standardized, and merged into a single dataframe for comparative analysis.

# 📌 Key Insights

- GDP positively influences happiness but is not the only determining factor.
- Social Support is one of the strongest contributors to happiness.
- Countries with higher Freedom generally report higher happiness.
- Healthy Life Expectancy shows a strong positive relationship with well-being.
- Low perceived corruption is associated with happier societies.
- Several countries outperform expectations despite lower GDP.
